Инструкция

1. Вставьте URL видео — Сначала скопируйте ссылку на видео с YouTube, которое хотите проверить на монетизацию. Вставьте её в поле для ввода на нашем сайте.

2. Нажмите кнопку "Проверить на монетизацию" — После ввода ссылки нажмите кнопку, чтобы инициировать проверку.

3. Просмотрите результат — Результат проверки будет отображён в поле под кнопкой.
